# LoLOS

A retro windows 95 styled WebOS with little personal portfolio, draggable windows, an interactive music player, and a functional notepad.

## ⚡ Quick Start

Since this is a web-based OS, there is no installation required! 
Just click the **[Live Demo](https://alinur7810.github.io/LoLOS/)** link above to boot up LoLOS directly in your browser.

---

## ✨ Features

- **Draggable Windows:** Drag windows around the desktop by their title bars, just like a real operating system.
- **Depth Management:** Clicking on any window automatically brings it to the front.
- **Music Player:** A retro media player with a working playlist, play/pause controls, and real audio playback.
- **Notepad:** A text editor where you can type notes using a retro typewriter font.
- **Dynamic Clock:** A classic taskbar with a real-time ticking clock.
- **Desktop Shortcuts:** Double-click desktop icons to open or reopen closed windows.
